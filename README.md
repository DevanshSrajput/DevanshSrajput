Loan Eligibility Predictor - Machine Learning Project
This project is a complete end-to-end machine learning application that predicts whether a loan application should be approved or rejected. It features a multi-page web interface built with Streamlit that allows for both live predictions and a detailed analysis of the model's performance.

Project Overview
The objective of this project is to build a reliable classification model to automate the loan eligibility process. By analyzing applicant details such as income, credit history, education, and marital status, the model provides an instant prediction with a confidence score. The application is designed to be user-friendly and can serve as a prototype for integration into fintech tools or loan-processing portals.

Features
Live Prediction Interface: A clean and simple form to enter applicant details and get an instant loan eligibility prediction.

Analysis Dashboard: A separate, detailed page showcasing the complete machine learning analysis, including:

Exploratory Data Analysis (EDA) visualizations.

Model performance metrics (Accuracy, AUC Score).

Confusion matrices and ROC curves for model comparison.

Multi-Model Training: The backend script trains both Logistic Regression and Random Forest models and automatically saves the best-performing one for deployment.

Modular Codebase: The project is structured into separate scripts for model training and the user interface, promoting clean and maintainable code.

Technical Stack
Language: Python 3.8+

Web Framework: Streamlit

Data Manipulation: Pandas, NumPy

Machine Learning: Scikit-learn

Data Visualization: Matplotlib, Seaborn

Model Persistence: Joblib

Setup and Installation
Follow these steps to set up and run the project on your local machine.

1. Clone the Repository
First, get the project files onto your machine.

git clone [https://github.com/your-username/loan-predictor.git](https://github.com/your-username/loan-predictor.git)
cd loan-predictor

2. Create a Virtual Environment (Recommended)
It's best practice to create a virtual environment to manage project dependencies.

# For Windows
python -m venv venv
venv\Scripts\activate

# For macOS/Linux
python3 -m venv venv
source venv/bin/activate

3. Install Dependencies
Install all the required libraries from the requirements.txt file.

pip install -r requirements.txt

4. Train the Model
Before launching the app, you must first run the training script. This will create the necessary model files.

python train_model.py

5. Run the Streamlit App
Now, you can launch the web application.

streamlit run app.py

Your web browser will automatically open to the application's URL (usually http://localhost:8501).

Machine Learning Workflow
The modeling process follows these key steps:

Data Preprocessing:

Handling Missing Values: Numerical columns are imputed with the mean, and categorical columns with the mode.

Feature Encoding: Categorical features are converted into numerical format using one-hot encoding.

Data Scaling: Numerical features are normalized using StandardScaler.

Model Training:

The data is split into an 80% training set and a 20% validation set.

Logistic Regression and Random Forest models are trained.

Model Evaluation:

The models are evaluated using Accuracy and AUC scores.

The train_model.py script automatically saves the best-performing model for the live application.
