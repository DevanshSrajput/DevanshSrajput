| Attribute | Details |

<details>
<summary><b>💸 NexusPay — Autonomous Budget-Aware AI Payment Agent</b></summary>
<br/>

> A reference implementation of an autonomous, budget-aware AI agent that buys its own data over the x402 HTTP payment protocol — planning paid data purchases, settling with testnet USDC, and synthesizing everything into a single answer.

| Attribute | Details |
|:----------|:--------|
| **Stack** | Python · Gemini · x402 Protocol · USDC (Testnet) · Agentic Planning |
| **Scale** | Multi-source paid data acquisition orchestrated per query |
| **Performance** | Query-level purchase planning minimizes spend per answer |
| **Security** | Hard per-query and daily budget caps — the agent cannot overspend |
| **Impact** | Working demonstration of machine-to-machine commerce, the frontier of agentic AI |
| **Repository** | [github.com/DevanshSrajput/NexusPay](https://github.com/DevanshSrajput/NexusPay) |

The agent receives a question, plans which paid data sources are worth buying (via Gemini), enforces strict budget constraints, executes each purchase as an x402 HTTP payment settled in testnet USDC, and fuses the acquired data into one synthesized response — a complete autonomous economic loop.

</details>

| Attribute | Details |
